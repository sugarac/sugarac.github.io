LeetCode 1. Two Sum

Posted on 2017-10-20
Words count in article:

Given an array of integers, return indices of the two numbers such that they add up to a specific target.
You may assume that each input would have exactly one solution, and you may not use the same element twice.
Example:
Given nums = [2, 7, 11, 15], target = 9,
Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].

Solution 1: HashMap O(n); O(n)
Build a HashMap between number and its index.
For each number n in the array:
Check if target - n exists in map:
If exists, return the indices.
Put n and its index in map.
Return {-1, -1} if not found.
